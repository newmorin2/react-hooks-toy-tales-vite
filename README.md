## Overview

A React application for managing a collection of toys. Users can view toys, add new toys, like toys, and donate (delete) toys using a mock backend API.


## Features

 **View toys**   <ul><li>View all toys on page load</li></ul> 
 **Add a toy**  <ul><li>Add a new toy</li></ul> 
 **Toy likes**   <ul><li>Increase toy likes</li></ul> 
 **Donate/remove toys**   <ul><li>Able to Donate/remove toys</li></ul> 
 **Fetching**  <ul><li>Backend communication using fetch</li></ul> 
  **Dynamic rendering**    <ul><li>Dynamic rendering using React state and props</li></ul> 



## Getting Started

### Prerequisites

This project requires the following:

- **Front-end language:** React
- **Server:** JSON Server
- **Scripting Language:** JavaScript


### Installation

Build HabitFlow from the source and install dependencies:

1. **Clone the repository:**

    ```sh
    ❯ git clone https://github.com/newmorin2/react-hooks-toy-tales-vite
    ```

2. **Navigate to the project directory:**

    ```sh
    ❯ cd react-hooks-toy-tales-vite
    ```

3. **Install the dependencies:**

  echo 'npm install'

4. **Start the backend server**
    ```sh
    ❯ json-server --watch db.json --port 3001
    ```
5. **Start the app**
    ```sh
    ❯ npm run dev
    ```

### Usage

Run the project with:

'npm run dev'

Run the test with:

'npm test'
